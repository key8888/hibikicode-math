# レッスン1: グラフを描いてみよう

ようこそ！この教材では **Python** と **Bokeh** を使って、一次関数や二次関数のグラフを描きながらプログラミングを学びます。

## 目標
- エディタに用意されたサンプルコードを実行する
- 実行結果のグラフを確認する
- グラフの色やタイトルを変更してみる

## やってみよう
1. 右側のエディタにあるコードを確認します。
2. 画面上部の「実行」ボタンを押しましょう。
3. 左の「グラフ表示」タブに描画された線を観察します。

### 練習
- `plot.title.text = "好きなタイトル"` を追加してタイトルを書き換えてみよう。
- `line_color` の値を変えて色を変更してみよう。
- `print` 文を追加して実行ログにメッセージを表示してみよう。

> ヒント: グラフが表示されない場合は、`plot = new_plot(...)` のように Bokeh の `figure` オブジェクトを `plot` 変数に入れているか確認しましょう。
