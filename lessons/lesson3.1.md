## 3.1 グラフ業界の専門用語

### 説明＆コード
グラフの世界では、いくつかのキーワードがある。ここを押さえれば、Bokehで描くときの会話がスムーズになるよ。

- **軸（じく）**: グラフの骨組み。横方向を $x$ 軸、縦方向を $y$ 軸と呼ぶ。Bokehでは `plot.xaxis` と `plot.yaxis` で設定できる。
- **座標（ざひょう）**: 点の位置を表す数字のペア。例えば $(2, 3)$ は「右に2、上に3」。
- **原点（げんてん）**: $x$ 軸と $y$ 軸が交わる点で、座標は $(0, 0)$。
- **凡例（はんれい）**: 色やマークが何を意味するかを説明するラベル。Bokehでは `legend_label` 引数に文字列を渡す。
- **マーカー**: 点を描くときの形。丸、四角、三角などを選べる。Bokehの `marker` 引数で指定する。
- **グリッド**: 背景にある薄い線。目盛り（めもり）を読みやすくしてくれる。

まずは空っぽのグラフを作って、これらの部品がどこにあるか確かめよう。`new_plot` は新しいキャンバスを作る関数だよ。

```python
plot = new_plot(title="グラフの部品チェック", width=500, height=350)

# 軸ラベルを付けておく
plot.xaxis.axis_label = "x軸 (横)"
plot.yaxis.axis_label = "y軸 (縦)"

# グリッドの見え方も調整可能
plot.xgrid.grid_line_dash = "dashed"  # 点線（破線）にする
plot.ygrid.grid_line_alpha = 0.3      # 透明度を下げて薄くする

# 凡例の位置を変えてみる
plot.legend.location = "top_left"
```

ここではまだ点を置いていないけれど、`plot` という変数がグラフの本体。あとで点や線を追加しても、同じ `plot` にどんどん情報を盛り込めるよ。
