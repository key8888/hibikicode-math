## 7.5 誤差（error）を見てみよう

### 説明＆コード
データ点が直線からどれだけズレているかを「誤差」と呼ぶ。誤差を見れば、線の良し悪しが分かるよ。まずは縦方向のズレを計算してみよう。

```python
x_vals = [0, 1, 2, 3, 4, 5, 6]
y_vals = [0.5, 1.0, 1.8, 2.7, 3.5, 4.2, 6.0]

# 仮の近似直線 y = 0.8x + 0.3

def line(x):
    return 0.8 * x + 0.3

errors = []
for x, y in zip(x_vals, y_vals):
    diff = y - line(x)  # 観測値 - 直線の値
    errors.append(diff)

print("各点の誤差:", errors)
print("誤差の絶対値の合計:", sum(abs(d) for d in errors))
```

出力された誤差を見て、プラスなら線より上、マイナスなら線より下にあることが分かる。絶対値の合計が小さいほど、線がデータに近いイメージ。
