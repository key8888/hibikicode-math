# 第6章 二次関数
## 6.1 関数を「動かす」という発想

### 説明＆コード
グラフは式を書き換えるだけで上下・左右・傾き・形が変わる。一次関数 \( y = ax + b \) を例に、「動かす」感覚をつかもう。これが二次関数の形を理解するための橋渡しになる。

- 上下に動かす：式に \(+k\) を足すと、グラフ全体が上に \(k\) だけ平行移動する。
- 左右に動かす：\(x\) を \(x-h\) に置き換えると、右に \(h\) だけ移動する。
- 傾きを変える：係数 \(a\) を大きくすると、線が急になる。

まずは初期状態の一次関数を描こう。

```python
plot = new_plot(title="関数を動かす感覚", width=600, height=400)
plot.xaxis.axis_label = "x"
plot.yaxis.axis_label = "y"

# 初期の一次関数 y = x + 1
plot_function(lambda x: x + 1, x_start=-5, x_end=5, target_plot=plot,
              line_color="#1f77b4", legend_label="y = x + 1")
plot.legend.location = "top_left"
```

このグラフを基準に、次のステップで動かしながら違いを確かめていくよ。
