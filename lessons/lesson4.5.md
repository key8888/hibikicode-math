# 第4章 一次関数
## 4.5 一次関数の応用問題（文章題→式→グラフ）

### 説明＆コード
文章題を「式」と「グラフ」に落とす流れを練習しよう。例として「基本料金200円＋1kmあたり50円のタクシー」を考える。

1. 文章を式にする：距離を \(x\) km、料金を \(y\) 円とすると
   \[
   y = 50x + 200
   \]
2. 式をグラフにする：`plot_function` で描いてみる。

```python
plot = new_plot(title="タクシー料金", width=600, height=400)
plot.xaxis.axis_label = "距離 x (km)"
plot.yaxis.axis_label = "料金 y (円)"

plot_function(lambda x: 50 * x + 200, x_start=0, x_end=10, target_plot=plot,
              line_color="#1f77b4", legend_label="料金の式")

# 具体例の点（3km 乗ったとき）
plot_points([3], [50 * 3 + 200], target_plot=plot, size=10,
            color="#d62728", legend_label="3km の料金")
plot.legend.location = "top_left"
```

式にすると「固定費＝200円」「距離ごとの追加＝50円」とすぐ分かる。グラフでも切片が200、傾きが50で右上がりになるね。

#### ミニ課題
自分で「基本料＋追加料」の例を1つ考えて、式を立ててグラフを描いてみよう。スマホ料金でもお店のキャンペーンでもOK！
