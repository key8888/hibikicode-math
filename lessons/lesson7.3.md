## 7.3 一次関数に当てはまりそう？ざっくり予測してみる

### 説明＆コード
散布図を見て、「直線を引いたら近そうか？」を目分量で考えてみよう。厳密な計算より、直感で傾きと切片を推測する練習だよ。

```python
x_vals = [0, 1, 2, 3, 4, 5, 6]
y_vals = [0.5, 1.0, 1.8, 2.7, 3.5, 4.2, 6.0]

plot = new_plot(title="直線で近そう？", width=600, height=400)
plot.xaxis.axis_label = "x"
plot.yaxis.axis_label = "y"

plot_points(x_vals, y_vals, target_plot=plot, size=10, color="#1f77b4", legend_label="データ")

# 目分量で引いた直線（例：傾き0.8、切片0.3）
plot_function(lambda x: 0.8 * x + 0.3, x_start=-1, x_end=7, target_plot=plot,
              line_color="#d62728", legend_label="目分量の線")
plot.legend.location = "top_left"
```

この線が「だいたい真ん中を通っていそうか？」を自分の目で判断しよう。ズレているなら、傾きや切片を少しずつ変えてみて。

#### ミニ課題
傾きを 0.5、1.0、1.2 に変えた線をそれぞれ描いてみて、どれが一番データに近いか友達に説明してみよう。
