## 1.3 学習の進め方とAIとの付き合い方

### 学習目標
- 予習→実験→振り返りの3ステップを、自分で回すイメージを持つ。
- 簡単なリストのデータで、学習の進み具合を可視化する体験をする。
- AIを使うときのマナー（著作権・情報セキュリティ）を押さえる。

## 説明＆コード
ラボの基本リズムは「予習 → 実験 → 振り返り」。難しいコードはいらないので、まずはこの流れを数字で記録してみよう。例として、1週間で3回サイクルを回したときの「自分の手応えスコア」を点にしてみる。

```python
# 学習サイクルごとのスコアを点で表す
plot = new_plot(title="学習サイクルの記録", width=550, height=380)

# サイクル番号（1回目、2回目、3回目）
cycle_number = [1, 2, 3]
# それぞれの回での理解度（0〜100 くらいで自分の感覚でOK）
understanding_score = [40, 65, 82]

# 点を打つだけで十分。右肩上がりなら、学習が進んでいるサイン。
plot_points(cycle_number, understanding_score, target_plot=plot, size=12,
            marker="triangle", color="#2ca02c", legend_label="理解度の記録")
```

「点を打つだけ」で、どの回で伸びたかが見える。線でつなぎたいときは、`cycle_number` を細かくしたり、次のように簡単な関数を用意して使えるよ。

```python
# ざっくりした傾向を線で描きたいときの例（オプション）
# y = 20x + 20 という一次関数を自分で決めてみたパターン

def trend(x):
    return 20 * x + 20

plot_function(trend, x_start=1, x_end=3, num=50,
              target_plot=plot, line_color="#1f77b4", legend_label="ざっくり傾向")
```

AIと付き合うときの大事なポイントは次の3つ。
- **著作権**: 他人の文章やコードをそのままコピーすると権利を侵害することがある。引用するときは出典を書く。
- **情報セキュリティ**: 名前・住所・連絡先などの個人情報は送らない。学校のテスト問題など守るべき情報も共有しない。
- **自分で検証する**: AIからもらった式やコードが本当に合っているか、計算やグラフで自分の目でもチェックする。
