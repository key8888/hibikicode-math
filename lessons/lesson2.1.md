## 2.1 変数とprint()

### 説明＆コード
Pythonで数を入れておく「箱」みたいな役割が変数だよ。変数には名前をつけて、値（数や文字など）を代入するんだ。たとえば $x=3$ みたいに書くと、箱$x$に3が入っているイメージ。

```python
# 数字の変数
number = 3
# 文字列（string）という文字のかたまり
message = "こんにちは"
```

`print()` は中身を画面に出すための命令。数学で言うと、計算した答えをノートに書く感じ。変数を作ってから `print()` すると確認できるよ。

```python
name = "光"
age = 15
print("名前:", name)
print("年齢:", age)
```

型（かた）というのは、入っている値の種類のこと。整数（`int`）、小数（`float`）、文字列（`str`）などがある。`type()` で確かめられる。

```python
value = 2.5
print(type(value))  # <class 'float'> と出てきたら小数型
```

Bokeh 環境でグラフを出したいときも、まず変数でデータを用意してから関数を呼び出すよ。ここでは単純に点を1つ表示してみよう。

```python
x = [0]
y = [2]
plot = new_plot(title="1点だけ表示してみる")
plot_points(x, y, target_plot=plot, legend_label="点A")
```

#### ミニ課題
