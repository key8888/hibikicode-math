## 7.1 データを見ると「関数の形」が見えてくる？

### 説明＆コード
いきなり式を書くより、まずデータを眺めて「直線っぽい」「曲線っぽい」を感じるところからスタート。散布図を見ながら、増加・減少・カーブの雰囲気を探ろう。

```python
# ざっくりしたデータ例
x_vals = [0, 1, 2, 3, 4, 5]
y_vals = [1, 1.2, 2.5, 3.8, 3.9, 6.2]

plot = new_plot(title="データの気配", width=600, height=400)
plot.xaxis.axis_label = "x"
plot.yaxis.axis_label = "y"

plot_points(x_vals, y_vals, target_plot=plot, size=8, color="#1f77b4", legend_label="データ")
plot.legend.location = "top_left"
```

点が右上がりで並んでいるけど、ところどころ曲がっている感じが分かる？こうした感覚が、三次関数の「S字っぽさ」にもつながるよ。
