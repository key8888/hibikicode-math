## 2.2 配列の概要

### 説明＆コード
配列（はいれつ）はデータをまとめて入れておく棚みたいなもの。Pythonでは `list`（リスト）と呼ばれて、`[]` で作るよ。例えば $A=\{2,4,6\}$ みたいな集合をコードにするとこうなる。

```python
scores = [2, 4, 6]
print(scores)
```

リストを使うと、複数の値をひとつの変数名で管理できる。身近な例だと、1週間の気温データをまとめるイメージ。

```python
week_temperatures = [18.2, 19.0, 20.5, 21.1, 19.8, 18.9, 17.5]
print("週間平均気温データ:", week_temperatures)
```

Bokeh で点をたくさん表示したいときも、`x` や `y` の値をリストで準備するよ。

```python
x_points = [0, 1, 2, 3]
y_points = [0, 1, 4, 9]  # $y = x^2$ の値
plot = new_plot(title="二乗のデータを点で表示")
plot_points(x_points, y_points, target_plot=plot, legend_label="y = x^2")
```
