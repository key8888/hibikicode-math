# 第4章 一次関数
## 4.2 傾きと切片をグラフから読み取る練習

### 説明＆コード
直線上の2点 \((x_1, y_1), (x_2, y_2)\) から傾きを求める公式は
\[
 a = \frac{y_2 - y_1}{x_2 - x_1}
\]
「上がり幅 ÷ 右に動いた幅」という意味。切片 \(b\) は $x=0$ での $y$ 値だから、グラフが $y$ 軸と交わる点を見ればよい。

簡単なデータをプロットして、そこから傾き・切片を考えてみよう。

```python
plot = new_plot(title="2点から傾きを考える", width=600, height=400)
plot.xaxis.axis_label = "x"
plot.yaxis.axis_label = "y"

# 2点を用意
x_points = [1, 4]
y_points = [2, 5]
plot_points(x_points, y_points, target_plot=plot, size=10, color="#d62728",
            legend_label="観察する2点")

# 2点を結ぶ一次関数を描画
plot_function(lambda x: (5 - 2)/(4 - 1) * x + (2 - (5 - 2)/(4 - 1) * 1),
              x_start=0, x_end=5, target_plot=plot,
              line_color="#1f77b4", legend_label="通る直線")
plot.legend.location = "top_left"
```

ここでは点 $(1,2)$ と $(4,5)$ を結んだ直線。計算すると傾きは \( a=1 \)、切片は \( b=1 \) になるよ。
