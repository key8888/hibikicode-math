# 第6章 二次関数
## 6.10 まとめ：関数は“動かせばわかる”

### 説明＆コード
この章では一次関数を動かしながら、平行移動・反転・伸縮の感覚をつかんだ。どれも式のほんの一部を書き換えるだけでグラフがスッと動く。二次関数でも同じ考え方で、頂点を動かしたり開きを変えたりできる。

最後に、平行移動と反転をまとめたシンプルなコードをもう一度載せておくよ。二次関数に置き換えれば応用できる。

```python
plot = new_plot(title="まとめの一枚", width=600, height=400)
plot.xaxis.axis_label = "x"
plot.yaxis.axis_label = "y"

# 基本の一次関数
plot_function(lambda x: x, x_start=-6, x_end=6, target_plot=plot,
              line_color="#1f77b4", legend_label="y = x")

# 頂点っぽく平行移動（右に2、下に1）
plot_function(lambda x: (x - 2) - 1, x_start=-6, x_end=6, target_plot=plot,
              line_color="#ff7f0e", legend_label="右に2 下に1")

# 反転（左右＋上下）
plot_function(lambda x: -( -x ), x_start=-6, x_end=6, target_plot=plot,
              line_color="#2ca02c", legend_label="両反転で元に戻る")

plot.legend.location = "top_left"
```

これらの操作は、二次関数 \( y = a(x-h)^2 + k \) の形で全部活躍する。動かして理解するクセをつければ、関数がもっと身近になるよ。
