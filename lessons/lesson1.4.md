## 1.4 グラフの色をえらぶコツ

### 学習目標
- `color` 引数に渡す文字列の意味（色名・Hex・RGB・HSV）を知る。
- `color="#1f77b4"` がどんな色か説明できる。
- 目的に合わせて色を変え、見やすいグラフを作る準備をする。

## 説明＆コード
グラフのわかりやすさは色選びで大きく変わるよ。`plot_points` や `plot_function` の `color`（または `line_color`）には、いくつかの書き方がある。

- **色名**: "red"、"blue" のような英語名。短く書ける。
- **Hex（16進数）**: `"#RRGGBB"` の形。`#1f77b4` はR=1f、G=77、B=b4という青っぽい色。
- **RGB**: `(r, g, b)` の3つ組。0〜255で色を指定（BokehではタプルでもOK）。例: `(31, 119, 180)` は `#1f77b4` と同じ。
- **HSV**: 色相(H)、彩度(S)、明るさ(V)で指定。慣れないうちはHexか色名から始めよう。

### いろいろな色を試すコード例
```python
plot = new_plot(title="色のサンプル", width=620, height=420)

x_list = [1, 2, 3, 4]

# 色名
plot_points(x_list, [1, 1, 1, 1], target_plot=plot, size=12,
            marker="circle", color="red", legend_label="red (色名)")

# Hexコード（#1f77b4 は落ち着いた青）
plot_points(x_list, [2, 2, 2, 2], target_plot=plot, size=12,
            marker="square", color="#1f77b4", legend_label="#1f77b4 (Hex)")

# RGBタプル（255が一番濃い、0が暗い）
plot_points(x_list, [3, 3, 3, 3], target_plot=plot, size=12,
            marker="triangle", color=(0, 180, 80), legend_label="(0,180,80) (RGB)")
```

`#1f77b4` のようなHexは、「1f（赤）・77（緑）・b4（青）」で表すよ。16進数なので、0〜9とA〜Fを使うことに注意（小文字でもOK）。
