# 第1章 数学ラボとは？
## 1.1 数学ラボとは？

### 学習目標
- hibikicode math lab で何ができるかを、やさしい言葉でイメージする。
- 一次関数 \(y = ax + b\) を、まず「数の変化」として感じ取る。
- 簡単なPythonコードで、計算結果を表にしてからグラフにする流れを体験する。

## 説明＆コード
数学ラボは「数字で考えて → 計算して → 見て確かめる」をくり返す場所だよ。最初から難しいグラフを描く必要はなし。まずは、式を

1. 文字に置き換える（例: 傾き \(a\)、切片 \(b\)）
2. 数字を入れて計算する
3. 計算した結果を表やグラフで眺める

という順番で試してみよう。

### ステップ1: 計算してみる
Pythonは「計算のメモ帳」みたいに使える。`x` に入れた値がどう変わるか、まずは表にしてみよう。

```python
# y = ax + b の形を、数字を入れて試す
# 傾き a と 切片 b をまず決める

a = 2  # 1 増えると 2 ずつ増える
b = 1  # x=0 のときの値

# 計算したい x のリストを用意
x_values = [-2, -1, 0, 1, 2]

# y を計算して、新しいリストに入れる
# 「for」を使うと、同じ計算を何回もくり返せるよ

y_values = []
for x in x_values:
    y = a * x + b
    y_values.append(y)
    print(f"x={x} のとき y={y}")
```

これで「xを1増やすとyが2増える」感覚がつかめるはず。`print` で出る表を眺めてみてね。

### ステップ2: グラフにしてみる
計算したリストをそのまま点として描くと、式とグラフがつながる。`plot_points` は「xのリスト」と「yのリスト」を渡すだけでOK。

```python
plot = new_plot(title="最初の一次関数グラフ", width=600, height=400)

plot_points(x_values, y_values, target_plot=plot, size=10,
            marker="circle", color="#1f77b4", legend_label="計算した点")
```

`plot_points` は「表を点にする」イメージ。線でつなぎたいときは、少し先のレッスンで `plot_function` を使うから、いまは点だけで十分。

#### ミニ課題
- `a` と `b` の数字を自分で変えて、`print` の表と点の位置がどう変わるかを確かめてみよう。

#### 振り返りチェックリスト
- [ ] \(y = ax + b\) の式に数字を入れて、計算→表→点描画の流れがイメージできた。
- [ ] `x_values` と `y_values` の長さが同じでないと、点が描けないことに気づいた。
- [ ] もっと知りたいこと、分からないことをメモした。

#### 発展アイデア
- `x_values` をもっと細かい値（例: -2 から 2 を0.5ずつ）にして、点がどんな線を作りそうか眺めてみよう。
- 身近な場面（歩く距離と時間、お小遣いの貯まり方）を \(y = ax + b\) に当てはめて、同じ手順で表と点を作ってみよう。
