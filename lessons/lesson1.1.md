## 1.1 数学ラボとは？

### 学習目標
- hibikicode math lab でどんな体験が待っているのか、全体像をつかむ。
- 数学の式 \(y = ax + b\) のような身近な関数が、プログラムでどう扱えるかをイメージする。
- グラフ描画用ヘルパー関数の雰囲気を知り、コードに苦手意識を持たない。

## 説明＆コード
数学ラボは「考えて → 試して → もう一度考える」を繰り返して、数字やグラフの意味を体で覚えていく場所だよ。中学で習う比例の式 \(y = ax\) や、高校で出てくる一次関数 \(y = ax + b\) を、自分の手で動かせるのがポイント。たとえば傾き \(a\) を変えると線の角度がどう変わるか、グラフを見ながら確かめられる。

ここでは環境に用意されている `new_plot` と `plot_function` をちょっとだけ触ってみよう。コードを実行すると、一次関数のグラフが描かれて、ラボでやることのイメージがつかめるはず。

```python
# 一次関数 y = 2x + 1 のグラフを描いてみる例
plot = new_plot(title="最初のラボ体験グラフ", width=600, height=400)

def line_func(x):
    # 傾きが2、切片が1の一次関数
    return 2 * x + 1

plot_function(line_func, x_start=-5, x_end=5, num=200, target_plot=plot,
              line_color="#1f77b4", legend_label="y = 2x + 1")

# 座標のイメージを持ちやすくするため、参考の点も表示
x_points = [-2, 0, 2]
y_points = [line_func(x) for x in x_points]
plot_points(x_points, y_points, target_plot=plot, size=10,
            marker="circle", color="#d62728", legend_label="通る点")
```

#### ミニ課題
- グラフの傾きを変えるには、`line_func` の中の数字をどう変えればいいかな？自分で \(a\) や \(b\) を変えて、線の角度や位置を確かめてみてね。

#### 発展アイデア
- 傾き \(a\) や切片 \(b\) を日常の変化（身長の伸び、お小遣いの貯まり方など）にたとえて、グラフで表現してみよう。
- 学校で習った他のグラフ（反比例や二次関数）も、同じように描けるのかを試すと面白いよ。
