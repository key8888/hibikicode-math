## 説明＆コード
hibikicode math lab では、ブラウザ上でPythonを書いてすぐにグラフが出せるセットがそろっているよ。ここで押さえておきたい道具をざっくり紹介するね。

1. **Pythonエディタ**: コードを書くノート。変数（値を入れておく箱）や関数（動作をまとめたもの）を作る場所。
2. **Bokeh描画エリア**: グラフや図形を表示するキャンバス。座標 \((x, y)\) に点を置いたり、式 \(y = f(x)\) の線を描く。ヘルパー関数があるから、難しい設定なしで描ける。
3. **AIアシスタント**: コードのヒントや数学の解説を、質問に答える形でもらえる相棒。ただし後で出てくるルールにしたがって、正しく活用するよ。

まずはツールを組み合わせたイメージを持つため、`plot_points` を使って簡単な散布図を作ってみよう。グラフ上の点は、数学で習う座標の考え方そのものだよ。

```python
# 教室の座席をイメージした散布図
plot = new_plot(title="教室の座席マップ", width=500, height=400)

# 列（x軸）と行（y軸）で座席を表す
seat_x = [1, 2, 3, 4]
seat_y_front = [1, 1, 1, 1]
seat_y_back = [2, 2, 2, 2]

plot_points(seat_x, seat_y_front, target_plot=plot, size=12,
            marker="square", color="#ff7f0e", legend_label="前の列")
plot_points(seat_x, seat_y_back, target_plot=plot, size=12,
            marker="circle", color="#1f77b4", legend_label="後ろの列")
```

コードを実行したら、`seat_x` や `seat_y_back` の値を変えて、自分のクラスの座席図っぽくしてみるのもアリ。これが「データを作って、すぐ可視化する」ラボの基本体験だよ。
